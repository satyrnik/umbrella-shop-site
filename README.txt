Ambrella (Vite + React + Tailwind)

1) Установи зависимости:
   npm install

2) Запусти локально:
   npm run dev

3) Открой адрес из консоли (обычно http://localhost:5173/)

Структура:
- src/AmbrellaV3.jsx — твой основной компонент (можешь править)
- src/App.jsx — обёртка
- src/index.css — Tailwind
- tailwind.config.js, postcss.config.js — конфиги
